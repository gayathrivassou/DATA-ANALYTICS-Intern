
# Task 3: SQL Basics – Filtering, Sorting, Aggregations

## Queries Included
1. **Basic SELECT** – Understand table structure and sample records.
2. **COUNT(*)** – Validate number of records after import.
3. **WHERE + ORDER BY** – Filter Technology category and find top-selling products.
4. **GROUP BY + Aggregations** – Calculate total sales, average profit, and order count per category.
5. **HAVING** – Show only categories with sales greater than 100,000.
6. **BETWEEN** – Generate monthly sales reports using date ranges.
7. **LIKE** – Search customers by name pattern.

## Output
- `sales_summary.csv` contains category-wise aggregated sales and profit data.

## Purpose
These queries demonstrate core SQL skills required for data analysis: filtering, sorting, aggregation, and business-focused reporting.
